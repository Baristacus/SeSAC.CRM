# 새싹 CRM 프로젝트

## 백엔드 API 설치 및 실행방법

1. 파이썬 가상환경 생성
2. `pip install -r requirements.txt`를 입력하여 패키지 설치
3. api폴더 내에서 `python main.py`를 입력하여 실행

### API 엔드포인트: http://127.0.0.1:5000/api

-   `/users`: 전체 유저 리스트
-   `/user/<user_id>`: 해당 유저의 상세 정보
-   `/stores`: 전체 스토어 리스트
-   `/store/<store_id>`: 해당 스토어의 상세 정보
-   `/items`: 전체 품목 리스트
-   `/item/<item_id>`: 해당 품목의 상세 정보
